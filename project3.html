Retro Multiplayer Game Console - Capstone Senior Design Project

This project is a **dual-screen gaming console** featuring **real-time multiplayer support** for two players. I led the **hardware design**, **PCB development**, and **game logic implementation** to create a seamless gaming experience. The console includes **classic games** like **Tetris**, **Pong**, and **Minesweeper**, and connects via **Bluetooth controllers**. The system also uses **VGA output** to drive two separate screens.

My Role: As the **Electrical Engineer**, I was responsible for designing the **main circuit board (PCB)**, ensuring **low-latency Bluetooth communication**, implementing the **VGA video output**, and managing **power distribution** to ensure up to **72 hours** of continuous gameplay.

Key Technologies: **ESP32**, **ATmega328P**, **Bluetooth (HC-05)**, **VGA Display**, **DC-DC Power Converters**, **Custom PCB Design**, **UART Communication**

GitHub Repository: All project code, hardware designs, and testing results are available on my GitHub repository.

Project Challenges & Achievements

  • Multiplayer Synchronization: Implemented **real-time synchronization** of game states between two **ESP32** microcontrollers over **UART**.
  • VGA Video Output: Designed and tested **VGA signal generation** for stable **dual-screen output** at **640x480 resolution**.
  • Bluetooth Communication: Optimized **Bluetooth communication** between the **controller** and the console to ensure **low-latency input** for smooth gameplay.
  • Power Management: Designed the **power system** to support **up to 72 hours of gameplay** using **lithium-ion batteries** and **DC-DC step-up converters**.
