Retro Multiplayer Game Console - Capstone Senior Design Project

Client: ECE 1896: Senior Design Project

Location: Pittsburgh, PA

Year: 2025

This project involved creating a **dual-screen multiplayer game console** with advanced functionalities, utilizing **embedded systems** and custom **hardware design**. The system runs **classic games** like **Tetris**, **Pong**, and **Minesweeper**, interfacing through **Bluetooth controllers** and delivering high-quality video via **VGA output** to two independent screens. As the **Electrical Engineer**, I spearheaded the design, implementation, and integration of **PCBs**, **power systems**, and **communication protocols**, all while adhering to rigorous engineering constraints and testing standards.

GitHub Repository: You can view the complete project details, including code, schematics, and documentation, on my GitHub repository.

Design Constraints & Challenges

  • Power Management: Ensured **72 hours** of continuous gameplay using **Li-ion batteries** and **DC-DC step-up converters**.
  • Low-Latency Bluetooth Communication: Optimized Bluetooth for **real-time interaction** with minimal input lag.
  • Hardware Integration: Integrated subsystems for **game logic**, **controller input**, and **VGA output** using **efficient signal routing**.
  • Multiplayer Synchronization: Achieved smooth synchronization using **UART communication** between two **ESP32 microcontrollers**.

Technical Design

  • Power System: The system uses **Li-ion rechargeable batteries** and a **DC-DC step-up converter** to ensure stable operation.
  • Bluetooth Communication: **HC-05 Bluetooth modules** enable low-latency communication between the controllers and console.
  • VGA Output Generation: **ESP32 microcontrollers** generate VGA signals for dual-screen output using **DMA (Direct Memory Access)** techniques.
  • Controller Design: The **controller PCB** integrates **ATmega328P microcontrollers** for wireless input and communication via Bluetooth.
